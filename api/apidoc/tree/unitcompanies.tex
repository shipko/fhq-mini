\section{Список подразделений}
\par

Для получения списка подразделений предприятия: \\
\begin{Verbatim}[frame=single]
http://<host>/api/tree/tree.php?company=1
\end{Verbatim}

В результате вернется вот такой вот список:
\begin{Verbatim}[frame=single]
{
    "result":"ok",
    "data":{
        "2":{
            "name":"Столярный цех"
        }
    }
}
\end{Verbatim}
